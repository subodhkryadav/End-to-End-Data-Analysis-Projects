# Excel Practice - Day 01

This exercise is focused on transforming a draft sales report into a cleaner and more usable Excel output. It is a practical introduction to working with raw tabular data and preparing it for reporting.

---

## Files Included

- **Report_Draft.csv** - Raw source report data
- **Report_Final .xlsx** - Final Excel workbook version

---

## Dataset Snapshot

The draft file contains order-level sales information with fields such as:

- Row ID
- Order ID
- Order Date
- Ship Date
- Ship Mode
- Customer Name
- Segment
- Region
- Category
- Sub-Category
- Product Name
- Sales
- Quantity
- Discount
- Profit

This makes the dataset suitable for reporting, filtering, formatting, and summary-based Excel practice.

---

## Objective

The goal of this task was to take a raw CSV report and convert it into a more refined Excel workbook that is easier to read, review, and analyze.

---

## Work Performed

- Opened and reviewed the raw report data
- Organized the columns into a structured table format
- Improved readability for reporting use
- Prepared the dataset inside an Excel workbook format
- Focused on making the report cleaner and more presentation-ready

---

## Skills Practiced

- Importing CSV data into Excel
- Understanding raw transactional report structure
- Cleaning and organizing worksheet data
- Preparing files for reporting and further analysis
- Building better spreadsheet presentation habits

---

## Learning Outcome

This Day 01 practice helped build a solid foundation in Excel reporting workflow by showing how raw exported data can be converted into a cleaner, business-friendly format.

---

This is an important first step before moving into formulas, summaries, pivot tables, charts, and dashboard-style Excel reporting.
