# Excel Practice

This folder contains my day-wise Excel practice work. Each subfolder represents a hands-on exercise designed to improve spreadsheet skills through real reporting and data-handling tasks.

---

## Practice Focus Areas

- Working with CSV and Excel files
- Cleaning and organizing raw datasets
- Formatting data for readability and reporting
- Preparing structured worksheets from draft data
- Building confidence with practical business-style tasks

---

## Learning Goal

The purpose of this practice section is to strengthen the Excel fundamentals that are essential in real-world data analysis roles. The work here is focused not just on formulas, but also on presenting information clearly and professionally.

---

## What You Will Find Here

- Raw draft files used as input
- Final workbook versions after cleaning or structuring
- Day-wise README files describing the task, workflow, and outcomes

---

Each day adds another layer of practical spreadsheet experience, from raw data preparation to polished business reporting.
